# Supermarket (Next.js)

A Next.js clone of the Retail Partner React frontend.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Assets

- **Logo**: A placeholder `public/logo.svg` is included. To use your original logo, replace it with `logo.png` from the React project and update `Navbar.tsx` and `Footer.tsx` to use `src="/logo.png"`.
- **Hero video**: Place `hero-video.mp4` in the `public/` folder to enable the hero background video. The hero will show a poster image until the video loads.

## Routes

- `/` - Home
- `/shop` - Product catalogue

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS 3
- Radix UI (Toast, Tooltip)
- Sonner (toast notifications)
- Lucide React (icons)
